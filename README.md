
# 💼 Developer Portfolio

A modern and responsive personal portfolio website built using **HTML5** and **CSS3**. This project showcases a clean dark-themed interface, smooth animations, and a professional layout for presenting projects, skills, and personal information.



## ✨ Features

- 🎨 Modern dark UI design
- 📱 Responsive layout
- 🚀 Professional hero section
- 💻 Developer illustration
- 📂 Project showcase section
- 🎯 Smooth hover effects
- 🌈 Consistent color palette
- 🔗 Social media links
- ⬆️ Back-to-top button
- ⚡ Clean and organized code

---

## 🛠️ Built With

- HTML5
- CSS3
- Google Fonts
- Boxicons

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/developer-portfolio-dark-theme.git
```

Open the project

```bash
cd developer-portfolio-dark-theme
```

Simply open **index.html** in your browser.
